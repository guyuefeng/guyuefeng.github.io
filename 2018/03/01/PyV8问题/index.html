sublime Text 2 - Error while loading PyV8 binary

sublime Text 2安装Emmet插件过程中，出现以下错误

通过以下步骤可以解决

1.转到对话框中的链接地址
2.进入路径 C:\Users\UserName\AppData\Roaming\Sublime Text 2\Packages后 创建 PyV8 这个文件夹
3.将C:\Users\UserName\AppData\Roaming\Sublime Text 3\Packages\PyV8 把下载的 pyv8-win64 解压 拷贝进来放入PyV8目录就好了.
4.重启sublime

注意下载的系统版本，64位的机器有可能需要pyv9-win32的包，需根据实际情况确定
