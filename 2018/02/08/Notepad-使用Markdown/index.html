Notepad++使用Markdown

发表于 2018-02-08

一、实现markdown语法高亮

首先下载Markdown https://github.com/Edditoria/markdown_npp_zenburn 规则文件，然后可以通过Notepad++中的语言–>自定义语言格式–>导入 来实现；也可以通过手动操作实现，点击 开始菜单–>运行–>输入 %APPDATA%\Notepad++,将userDefineLang.xml放置到此目录，重启软件。

二、实现markdown实时预览

下载NppMarkdown插件，复制dll到Notepad++安装目录的plugins目录下，重启软件
