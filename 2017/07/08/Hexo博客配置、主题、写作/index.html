Hexo博客配置、主题、写作

发表于 2017-07-08

今天说明一下Hexo博客基本配置和主题更换

一、Hexo博客整体配置

博客的整体配置在 E:\hexo\blog_config.yml文件中进行。默认使用的主题是landscape。更改_config.yml中内容使其生效。

#Site
title: 起今知行 | 思考者 #填写网站名，会在标签页上显示
subtitle: 自我管理，知识管理，时间管理，阅读，语音写作，思维导图，Wiz，TimeMeter #副标题，网站名下面
description: 知识管理，时间管理，自我管理，阅读，语音写作，思维导图，Wiz，TimeMeter #网站描述，便于搜索引擎用关键词检索
author: guyuefeng #作者
language: zh-CN #语言，中文

二、更换主题

1.安装主题

在博客目录E:\hexo 下右键点击 Git Bash,输入以下命令。其他的主题也类似操作。

git clone https://github.com/wuchong/jacman.git themes/jacman

2.启用主题

修改博客目录E:\hexo_config.yml 中的theme属性，将其修改为jacman。

theme:jacman

3.更新主题

在主题更新之前，请备份好主题目录下的_config.yml文件，后面会修改很多配置

cd themes/jacman
git pull origin master

到这里主题的更换就好了。

三、主题的配置

主题的配置在E:\hexo\themes\jacman_config.yml文件里面

menu: #导航栏
主页: /
归档: /archives
关于: /about #介绍一下你自己
网络工具: /tools #自己添加
资源共享: /share #自己添加

导航栏添加新页面

hexo new page “share”
hexo new page “tools”

四、写作

博客基本配置完成，主题也更换了，如何新写一篇文章。可以使用 hexo n 命令新建文章，也可以在E:\hexo\source_posts中新建一个后缀为.md的文件，使用NotePad++采用MarkDown来书写。

1.添加文章抬头信息

Hexo 默认新建文章抬头有title、date等属性，可能缺乏categories和meta标签。可以打开E:\Hexo\scaffolds\post.md文件中添加

title: #文章标题
date: #时间，一般不用改
categories: #目录分类
tags: #标签，格式可以是[Hexo,time]，中间用英文逗号分开
keywords: #文章关键词，多个关键词用英文打开

2.文章图片存放

想要文章中插入图片，按照Markdown语法来插入，格式为 。图片存放两种方式：在本地 E:\Hexo\source目录下新建一个存放图片文件夹，如：images,然后把 想要插入的图片放在里面，插入图片路径；第二种方式把图片上传至网络，然后插入图片路径。推荐使用第二种。
