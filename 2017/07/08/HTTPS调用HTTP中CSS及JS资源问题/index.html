发表于 2017-07-08

HTTPS调用HTTP中资源问题

HTML中添加外部CSS文件和外部js文件到Header和Footer，加载HTTPS页面时，一些浏览器会直接屏蔽，加载不了。

有没有通过HTTPS页面让浏览器加载外部内容方便的解决办法。

一般的调用方式指定使用HTTP方式

通用解决方案，浏览器会自动根据当前是HTTPS还是HTTP来给资源URL补上协议头的，可以达到无缝切换

如果是站内资源，最好HTTP和HTTPS各一份，再通过相对路径的方式引用，这样就可以完美的解决掉问题
