Sublime Text2支持浏览器预览设置

Sublime编辑HTML缺少了浏览器预览的功能，通过设置可以实现: view in browser(ctrl+b)。

步骤一

通过 Tools->Build System->New Build System ，编辑untitled.sublime-build文件

步骤二

+untitled.sublime-build 中设置以下内容，保存untitled.sublime-build 为需要的名称

+{
“cmd”: [“explorer.exe”,”$file”]
}

步骤三

+Save it in Package->User directory and switch the default build system

+设置完成之后，按Ctrl+B 即可用你的默认浏览器浏览HTML代码

注意：如果文件名是中文的预览会失败，保证HTML文件在英文目录下且文件名为英文
